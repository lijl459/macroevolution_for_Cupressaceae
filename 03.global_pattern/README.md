- `Cupressaceae_185sp_locations.csv`  
  Species occurrence data

## Scripts

- `check_site_per_sp.r`  
- `filter_decimals.r`  
- `filter_site.R`  
- `Global_diversiy.r`  
- `OLS-5.r`
