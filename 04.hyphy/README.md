# Molecular evolution analyses (HyPhy)

This directory contains scripts for analyzing episodic diversifying selection.

## Description

Scripts are used to process and summarize HyPhy outputs (aBSREL, BUSTED).

## Key scripts

- `parse_absrel.py`  
- `summarize_absrel_result.py`  
- `count_selected_by_branch_from_json.py`  
- `filter_busted_from_log.py`  

## Functionality

- Parse HyPhy output files
- Identify branches under selection
- Summarize gene-level results

## Notes

Requires HyPhy output files as input.
