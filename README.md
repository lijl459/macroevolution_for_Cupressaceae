# Macroevolutionary analyses of Cupressaceae

This repository contains scripts and data used in the study:

"Uneven diversification in Cupressaceae arises from ecological opportunity, trait syndromes, and episodic molecular change"

## Overview

This project integrates phylogenomics, diversification analyses, global biodiversity patterns, and molecular evolution analyses to investigate uneven diversification in Cupressaceae.

## Directory structure
1. Phylogenomic reconstruction (`01.phylogeny`)
2. Diversification analyses (`02.hisse`)
3. Global biodiversity analyses (`03.global_pattern`)
4. Molecular evolution analyses (`04.hyphy`)
